# 33slona company test task
This is a tesk task build with angular.js 1.6

## Required environments
- Node.js v.4+
- Gulp

## Installation
1) Clone repository
```bash
$ git clone git://github.com/qramilq/33slona.git
```
2) Install dependencies
```bash
$ npm install
```
## Run production mode
```bash
$ gulp run
```
## Run develop mode
```bash
$ gulp serve
```
